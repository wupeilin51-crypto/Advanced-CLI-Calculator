# Advanced CLI Calculator
A command-line calculator that supports multiple operations, including smart division modes and formatted output.

## Description
A simple CLI calculator in Python that supports basic operations and smart division options.

## How to Run
[Click here to download: Eileen Wu (CLI).py](https://github.com/user-attachments/files/24723127/Eileen.Wu.CLI.py)

## Features
- Supports operations: +, -, *, /, **
- Decimal or whole-number division modes
- Optional remainder display for whole division
- Handles division by zero safely
- Displays full equation and result
- Clean CLI interaction
